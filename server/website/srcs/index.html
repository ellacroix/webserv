WEBSERV

Welcome to the webserv demonstration website

It is a static website designed to illustrate the features of the webserv program.

Webserv is a 42school group project. It requires the students to implement a HTTP/1.1 web server running locally, able to serve a fully static website.

You can find the subject of the project here.

A web server is basically a program running on a computer, receiving requests from clients (typically web browsers) and sending content (typically files) as a response to the requesting client.

We will cover the following:

  1. HTTP basics
  2. webserv basics
  3. GET method
  4. POST method
  5. DELETE method
  6. Configuration
  7. CGI
